Installation

We are in the middle of rapid development at the moment so the best way to stay on top of all our updates is to pull or download straight from our git repository!

Cloning Git Repository Using SSH
  1. git clone git@github.com:codazen/formulationjs.git
  2. npm install
  3. npm run build-sample1
  4. Create a new tab in console
  5. npm start
  6. Navigate to formulationjs/samples/sample1/public/index.html
  7. Enjoy!

Form Definition

Build your forms using JSON! Refer to formulationjs/samples/sample1/src/forms/formDef.json

formDef.json
Form
Field Name Field Type Description
name string The name of the form. Displays at the top of the form.
body string The description of the form. Renders below the form name.
submitDisabled boolean Allows users to disable the submit button.
elements Object Array List of form elements that will be rendered in the form.
Element Object. Some objects have other fields specific to their component.
Field Name Field Type Description
type string The type of form element we want to render. Currently supports 'textbox', 'checkbox', or 'dropdown'.
data Object Attributes to render our form element. Object will vary depending on the element type.
Textbox Data Object
Field Name Field Type Description
id string A unique id for the text input.
name string Name attribute for the text input.
value string The initial value of the text input.
label string A label rendered adjacent to the text input.
required boolean A user must fill out this component in order to submit the form.
maxlength string Maximum characters allowed in the component. Should represent an integer value.
textboxState boolean Disables the textbox from being used.
email boolean Users must enter valid email addresses.
Textarea Data Object
Field Name Field Type Description
id string A unique id for the text input.
name string Name attribute for the text input.
value string The initial value of the text input.
label string A label rendered adjacent to the text input.
required boolean A user must fill out this component in order to submit the form.
maxlength string Maximum characters allowed in the component. Should represent an integer value.
textareaState boolean Disables the textbox from being used.
Checkbox Data Object
Field Name Field Type Description
name string Name attribute for the checkbox input.
value boolean The initial value of the checkbox input.
label string A label rendered adjacent to the checkbox input.
required boolean A user must fill out this component in order to submit the form.
options Object Array An array of the different options in the checkbox element.
Checkbox Options Object
Field Name Field Type Description
name string Text that will be displayed for a checkbox option.
value string | int The actual value passed when selecting this option.
Dropdown Data Object
Field Name Field Type Description
options Object Array An array of the different options in the dropdown element.
required boolean A user must fill out this component in order to submit the form.
value string | int The initial value of the dropdown element. Any option that matches this value will be selected by default when the dropdown renders.
Dropdown Options Object
Field Name Field Type Description
name string Text that will be displayed for a dropdown option.
value string | int The actual value passed when selecting this option.
Datepicker Data Object
label string A label rendered adjacent to the checkbox input.
value boolean The initial value of the datepicker input.
dateFormat string Allows user to setup specific date format.
required boolean A user must fill out this component in order to submit the form.
options Object Array An array of the different options in the checkbox element. 
Example JSON Form

{
  "name": "Sample 1 Form",
  "body": "This will be a description of the form.",
  "submitDisabled": false,
  "elements": [
    {
      "type": "textbox",
      "data": {
        "id": "id",
        "name": "Name",
        "value": "",
        "label": "Label",
        "required": true,
        "maxlength": "10",
        "textboxState": true
      }
    },
    {
      "type": "textbox",
      "data": {
        "email": true,
        "id": "id",
        "name": "Name",
        "value": "",
        "label": "Email",
        "textboxState": true
      }
    },
    {
      "type": "textarea",
      "data": {
        "id": "id",
        "name": "Name",
        "value": "",
        "label": "Label",
        "required": true,
        "maxlength": "300",
        "textareaState": true
      }
    },
    {
      "type": "dropdown",
      "placeholder":"Select...",
      "data": {
        "label": "Dropdown Label",
        "name": "Name",
        "options": [
          {
            "name": "Option 1",
            "value": 1
          },
          {
            "name": "Option 2",
            "value": 2
          }
        ],
        "value": "",
        "required": true
      }
    },
    {
      "type": "checkbox",
      "name": "Name",
      "data": {
        "value":[],
        "groupLabel": "Checkbox group label",
        "required": true,
        "options": [
          {
            "name": "Option 1",
            "value": 1
          },
          {
            "name": "Option 2",
            "value": 2
          }
        ]
      }
    },
    {
      "type": "datepicker",
      "data": {
        "label": "Please select a due date",
        "value": null,
        "dateFormat": "DD/MM/YYYY",
        "placeholderText": "dd/mm/yyyy",
        "maxlength": "10",
        "required": "false",
        "range": "false"
      }
    }
  ]
}
